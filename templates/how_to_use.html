Getting Started

README Emoji Converter helps you enhance your GitHub README files with relevant emojis to make them more engaging and expressive. Follow these simple steps:

Step 1: Upload Your README

On the homepage, upload your existing README.md file using the file selector or by dragging and dropping it.

Step 2: Customize Your Emojis

After uploading, you'll be taken to the customization page where you can:

  • Select multiple emojis for each header level (H1, H2, H3)
  • Choose emojis for your list items
  • Add your own custom emojis by typing them into the text fields

The tool will automatically randomize from your selected emojis for each element in your README.

Step 3: Apply and Download

Click "Apply Emojis" to see a preview of your enhanced README side-by-side with your original. If you like the result, click "Download Updated README" to save your emoji-enhanced file.

Features

  • Multiple emoji selection for each element type
  • Custom emoji input support
  • Smart randomization from your selected emojis
  • Preview before downloading
  • Special keyword emoji mapping for words like "important," "note," "warning," etc.
